## 🌦️ WeatherNow — Live Weather App

WeatherNow is a **live weather forecasting application** built with **React**.  
It displays real-time weather updates and automatically changes the background according to current weather conditions.

✔ Clean modern UI  
✔ Accurate API-weather data  
✔ Dynamic backgrounds for real weather moods  
✔ Fully responsive design  
✔ Easy and intuitive interface
